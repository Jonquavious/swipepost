# SwipePost

Tinder-style content approval app for social media managers.

## Features
- Swipe right to approve, left to reject, up to save
- Multi-account support
- Auto-play video preview
- Queue management

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Jonquavious/swipepost)

## Environment Variables
- `NEXT_PUBLIC_SUPABASE_URL` - Your Supabase project URL
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` - Your Supabase anon key

## Local Development
\`\`\`bash
npm install
npm run dev
